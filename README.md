# Site Crema di Latte

Site premium, elegante e responsivo para a sorveteria artesanal Crema di Latte, localizada em Brasília.

## Características

- **Design Premium**: Esquema de cores preto (#000000) e dourado metálico (#D4AF37)
- **Totalmente Responsivo**: Adapta-se perfeitamente a desktop, tablet e mobile
- **Animações Suaves**: Efeitos de hover e transições elegantes
- **SEO Otimizado**: Otimizado para buscas locais em Brasília
- **Navegação Suave**: Scroll suave entre seções
- **WhatsApp Flutuante**: Botão de contato sempre visível

## Seções do Site

1. **Home**: Seção hero com imagem de fundo e call-to-action
2. **Nosso Sabor**: Apresentação dos diferenciais com ícones elegantes
3. **Nossas Lojas**: Informações das lojas Asa Norte e Asa Sul
4. **Eventos**: Serviços para casamentos, aniversários e eventos corporativos
5. **Avaliações**: Depoimentos de clientes com estrelas douradas
6. **Contato**: Links para WhatsApp, Instagram e TikTok

## Tecnologias Utilizadas

- HTML5 semântico
- CSS3 com Flexbox e Grid
- JavaScript vanilla
- Google Fonts (Playfair Display e Poppins)
- Ícones SVG personalizados

## Como Visualizar

1. Extraia os arquivos do ZIP
2. Abra o arquivo `index.html` em qualquer navegador moderno
3. O site funcionará localmente sem necessidade de servidor

## Como Hospedar

### Opção 1: Hospedagem Gratuita (GitHub Pages, Netlify, Vercel)
1. Faça upload dos arquivos para o serviço escolhido
2. Configure o domínio personalizado se desejar

### Opção 2: Hospedagem Tradicional
1. Faça upload dos arquivos via FTP para seu servidor
2. Configure o domínio para apontar para a pasta do site
